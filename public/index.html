How to use in iPhone Shortcuts
  1. Copy a webhook URL above with Copy URL.
  2. Open the Shortcuts app and tap + to create a new shortcut.
  3. Add the action Get Contents of URL and paste the copied URL. Leave the method as GET.
  4. Name the shortcut something you can say, e.g. Play Random Jazz.
  5. Optional: Add to Home Screen for a one-tap icon, or just say “Hey Siri, Play Random Jazz.”

Each time you run it, a fresh random album plays in your chosen zone.
